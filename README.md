# Spatial_Flu_COVID_RSV

Spatial analysis of Influenza, COVID, and RSV spread from 2022-2026 using emergency department visit data from the CDC. Dataset link: https://data.cdc.gov/Public-Health-Surveillance/NSSP-Emergency-Department-Visit-Trajectories-by-St/rdmq-nq56/about_data

Work performed at the NIH's Division of International Epidemiology and Population Studies (DIEPS).
